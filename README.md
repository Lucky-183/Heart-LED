### 创意灵感的火花蔓延成火海，隐约浮现它该有的样子。

<br>

#### 硬件功能：
- 人体感应：检测到人体时，自动执行设定参数中的亮度序列。
- 可调亮度：支持255档LED亮度调节。
- 环境光自适应(alpha)：可根据环境灯光自动调节LED亮度。
- 蓝牙调参：可通过蓝牙对Heart LED进行无线控制，设定模式以及参数。
